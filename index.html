Program Learning Outcomes

The M.S. in Applied Data Science is built around six learning outcomes. Each one represents a fundamental skill a practicing data scientist needs to turn raw information into decisions that matter. Below is each outcome paired with a plain-language explanation of what it means in practice.

  1. Collect, store, and access data by identifying and leveraging applicable technologies. Knowing how to find the right data, choose the right place to keep it (a database, data lake, file system), and pull it out efficiently when it's needed.
  2. Create actionable insight across a range of contexts (e.g., societal, business, political), using data and the full data science life cycle. Walking a problem all the way from a vague question through cleaning, modeling, and interpretation to a recommendation a decision-maker can act on.
  3. Apply visualization and predictive models to help generate actionable insight. Using charts, dashboards, and statistical or machine learning models as tools to surface patterns and forecast what's likely to happen next.
  4. Use programming languages such as R and Python to support the generation of actionable insight. Writing reproducible code that handles data wrangling, analysis, and modeling so that work can be trusted, scaled, and shared.
  5. Communicate insights gained via visualization and analytics to a broad range of audiences (including project sponsors and technical team leads). Translating technical findings into language and visuals that a non-technical executive and a senior engineer can both understand and act on.
  6. Apply ethics in the development, use, and evaluation of data and predictive models (e.g., fairness, bias, transparency, privacy). Recognizing when data or models can cause harm, and building safeguards around fairness, consent, and responsible use.
