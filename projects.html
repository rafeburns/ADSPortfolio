IST 707 — Applied Machine Learning

Mental Health & Wellbeing Analysis: Predicting Anxiety and Depression Among Syracuse Students

Analyzed responses from 4,957 Syracuse University students in the 2025 Campus Wellness Survey to predict anxiety and depression using machine learning. Built Random Forest classifiers achieving 72.55% accuracy for anxiety and 79.04% for depression, then used DBSCAN and HDBSCAN clustering to segment students into distinct intervention profiles. Key finding: loneliness was the single strongest predictor of both conditions across all models, with lonely students showing 2–5× higher rates of mental health issues compared to socially connected peers.

Stakeholder
Syracuse University Barnes Center for Health & Wellbeing
Actionable Insight
Identified high-risk student clusters to support proactive, data-informed outreach — shifting from reactive crisis management
Skills Used
  • Python
  • scikit-learn
  • Random Forest
  • DBSCAN
  • HDBSCAN
  • t-SNE

IST 691 — Deep Learning in Practice

Indie Music Recommendation System

Built a convolutional autoencoder that compresses 5-second audio snippets into 256-dimensional embeddings, enabling music discovery from the kind of short clips people hear on TikTok or Instagram Reels. Trained on 83,290 chunks derived from 8,000 Free Music Archive tracks, the model reached a validation loss of 0.0042 MSE and achieved 70% Top-10 song-level accuracy. Even when the exact track wasn't retrieved, recommendations frequently surfaced same-genre songs — evidence that the embeddings captured meaningful acoustic similarity, not just memorized tracks.

Stakeholder
Independent artists and listeners looking to discover indie music from short audio clips
Actionable Insight
Short snippets contain enough signal for meaningful similarity matching — offering indie artists a discovery path that doesn't depend on existing popularity or streaming behavior
Skills Used

  • Convolutional Autoencoder
  • Mel Spectrograms
  • STFT
  • CNN
  • GPU Training

IST 615 — Cloud Management

Cloud-Deployed ML Workbench

Designed and deployed a full-stack web application that lets users train machine learning models without writing code. Users upload a CSV through the web frontend, and the backend returns summary statistics, trains models (linear and logistic regression, Random Forest, LightGBM, XGBoost), and runs predictions on new data. Trained models can be exported as .pkl files for reuse, and new CSVs can be uploaded to test or generate predictions against a saved model. Backend auto-deploys from GitHub to an Azure Web App; frontend is hosted on Vercel.

Stakeholder
Analysts and students who need to explore data and train baseline ML models without setting up a local Python environment
Actionable Insight
Demonstrated that a standard ML workflow — exploration, training, evaluation, export, and prediction — can be delivered as a lightweight cloud service, lowering the barrier for non-programmers to run real models
Skills Used

  • Vercel
  • Python
  • scikit-learn

  • XGBoost
  • REST API
